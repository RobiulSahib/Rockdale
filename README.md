# Rockdale Construction & Real Estate

Rockdale is a modern, responsive web application for a construction and real estate company. This repository contains both the frontend and backend components of the project.

## Project Structure

- `rockdale-updated-master/`: The frontend application built with React, Vite, GSAP, and Styled Components.
- `backend/`: The backend server built with Node.js, Express, and MongoDB.

## Getting Started

### Prerequisites

- Node.js (v18.0.0 or higher)
- npm, pnpm, or yarn
- MongoDB (local instance or MongoDB Atlas)

### Frontend Setup

1. Navigate to the frontend directory:
   ```bash
   cd rockdale-updated-master
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Backend Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file based on the environment requirements (MongoDB URI, Port, etc.).
4. Start the server:
   ```bash
   npm start
   ```

## Technology Stack

### Frontend
- **React 19**
- **Vite**
- **GSAP & Framer Motion** (Animations)
- **Styled Components & Bootstrap** (Styling)

### Backend
- **Node.js & Express**
- **MongoDB & Mongoose**
- **JWT & Bcryptjs** (Authentication)
- **Nodemailer** (Email notifications)

## Deployment

The project is configured for deployment on:
- **Netlify/Vercel** (Frontend)
- **Render/Heroku** (Backend)
